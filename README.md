# Matershinik Bot 🤖

Telegram-бот для язвительных комментариев на стикеры с использованием нейросети OpenRouter.

## Возможности

- Распознаёт стикеры в группах, супергруппах и личных чатах
- Генерирует остроумные, саркастичные ответы через GPT-3.5-turbo
- Работает 24/7 на Railway.app

## Технологии

- Python 3.10+
- aiogram 3.x (асинхронный фреймворк для Telegram)
- OpenRouter API (доступ к GPT-3.5-turbo)
- python-dotenv (управление переменными окружения)
